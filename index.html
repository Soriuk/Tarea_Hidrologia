Para una mejor visualizacin de los datos se adjudicaron nombres a las variables:

names(inp) <- c("fecha", "Estrella", "Banano")
attach(inp)
Tempdate <- strptime(inp[,1], format = "%d/%m/%Y")
MAQ_Estrella <- tapply(Estrella, format(Tempdate, format = "%Y"), FUN=sum)
MAQ_Banano <- tapply(Banano, format(Tempdate, format = "%Y"), FUN=sum)
write.csv(rbind(MAQ_Banano, MAQ_Estrella), file = "MAQrbind.csv")

El cdigo anterior fue utilizado para darle un formato a los datos del archivo y as poder crear los MAQ (Mean annual streamflow), adems con el cdigo "write.csv se cre un archivo para poder acceder fcilmente a los datos de cada MAQ con alguna herramienta, ejemplo: Excel.

Acumulados mensuales

MMQ_Estrella <- tapply(Estrella, format(Tempdate, format = "%m"), FUN=sum)
MMQ_Banano <- tapply(Banano, format(Tempdate, format = "%m"), FUN=sum)

Este cdigo es muy parecido al utilizado en los MAQ, simplemente se cambio su formato de ao (%Y) a mes (%m), teniendo las precauciones de las minsculas y adems cambiando el nombre a MMQ (mean monthly streamflow).
